## npm源:
常用源地址：  
官方： http://www.npmjs.org    
阿里： https://registry.npmmirror.com

查看源地址
```sh
npm config get registry
```

设置源地址：
```
npm config set registry http://www.npmjs.org

npm config set registry https://registry.npmmirror.com
```
